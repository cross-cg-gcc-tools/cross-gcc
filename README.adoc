= Cross libgccjit compilers for use with rustc_codegen_gcc

== Dependencies

This requires the https://github.com/antoyo/gcc-build-tools/tree/dev[dev branch of my gcc-build-tools fork].

== Build instructions

1. Install the dev branch of my fork of `gcc-build-tools`.
2. Build the compiler: `path/to/gcc-build-tools/build-gcc -a $ARCH -v master` (replace $ARCH by your target from `configs/targets/`, for instance `m68k`).
4. The compiler will be in `$HOME/xgcc/cross-tool-m68k-gccmaster/bin`.
